# OpenSpec 配置設定

查看和修改 OpenSpec 的配置：

// turbo
1. 查看目前配置：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec config
```

2. 查看專案配置檔：
```bash
cat /Users/rexhsu/Documents/type-exercise/openspec/config.yaml
```

3. 修改全域配置：
```bash
openspec config --global
```

4. 配置項目包括：
   - 預設 schema
   - 語言設定
   - 輸出格式
