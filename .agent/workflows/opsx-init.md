# OpenSpec 初始化

在新專案中初始化 OpenSpec：

1. 初始化 OpenSpec：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec init
```

2. 這會建立以下目錄結構：
   - `openspec/config.yaml` - 配置檔
   - `openspec/specs/` - 規格目錄
   - `openspec/changes/` - 變更提案目錄

3. 查看初始化結果：
```bash
ls -la /Users/rexhsu/Documents/type-exercise/openspec/
```
