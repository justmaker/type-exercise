# OpenSpec 狀態查看

執行以下步驟查看 OpenSpec 的狀態：

// turbo
1. 列出所有變更提案：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec list
```

// turbo
2. 列出所有規格：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec list --specs
```

3. 顯示變更的詳細狀態（如果有進行中的變更）：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec status
```
