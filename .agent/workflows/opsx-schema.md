# OpenSpec Schema 管理

管理 OpenSpec 的工作流程 Schema（實驗性功能）：

// turbo
1. 列出可用的 Schema：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec schemas
```

2. 查看 Schema 模板路徑：
```bash
cd /Users/rexhsu/Documents/type-exercise && openspec templates
```

3. Schema 定義了變更提案的工作流程結構：
   - 需要哪些 artifact (proposal, design, tasks, specs)
   - 各個 artifact 的模板
   - 驗證規則

⚠️ **注意**: Schema 功能目前是實驗性的
